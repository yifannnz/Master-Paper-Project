\chapter{基于结构保持的非配对HRTEM图像质量提升}
高质量的HRTEM图像对于在原子尺度上建立材料结构与性能之间的关系至关重要。然而，在实际实验中，为了捕获动态过程的高时间分辨率图像，往往需要在低剂量成像条件下进行采集，这不可避免地导致HRTEM图像质量严重退化，表现为原子排列模糊、结构信息不完整等问题，从而极大地限制了后续的精确结构分析能力。

为了解决上述问题，本章提出了一种基于结构保持的非配对HRTEM图像质量提升方法HRTEM-GAN。该方法采用循环一致性生成对抗网络（CycleGAN）框架，在非配对训练条件下训练，通过图像块级别的分布建模实现高质量域和低质量域之间的双向映射。与传统的全图翻译方法不同，本方法采用图像块级建模策略，缩小了生成的有效感受野范围，使网络能够聚焦于细粒度的原子结构特征，从而缓解了全图生成中常见的原子恢复不完整问题。为了更好地建模长程原子关联，本章在生成器的瓶颈层引入了视觉Transformer（ViT）模块，使其能够捕获超越局部卷积邻域的全局依赖关系。此外，本章明确引入了频域建模和约束机制，通过专用的频域分支和频域感知损失函数来利用HRTEM图像固有的频谱特性，在增强图像质量的同时保持原子尺度的结构保真度。

在算法性能验证与实验结果分析环节，本章在真实实验数据集上进行了广泛的定性和定量评估。通过与现有代表性方法进行对比分析，验证了所提方法在图像恢复质量和下游原子柱识别性能上的显著提升。实验结果表明，HRTEM-GAN不仅能够有效提升低质量HRTEM图像的视觉质量，还能显著提高原子柱识别的准确率，为后续的结构分析和材料性能研究提供了更可靠的数据基础。

本章的主要内容安排如下：首先介绍非配对HRTEM图像质量提升的问题背景和挑战；随后详细阐述基于结构保持的改进循环式生成网络的设计，包括空间频域联合特征提取的生成网络设计、基于自注意力机制的全局原子结构一致性建模，以及基于特征函数与频域一致性的结构保持约束设计；最后通过在真实实验数据集上的广泛定性和定量评估，验证所提方法的有效性，并与现有代表性方法进行对比分析，证明本方法在图像恢复质量和下游原子柱识别性能上均取得了显著提升。

针对低剂量成像条件下HRTEM图像退化严重且难以获得严格配对训练数据的问题，本章节提出了一种基于结构保持的非配对图像质量提升框架HRTEM-GAN，其整体流程如图\ref{fig:3-1}所示。该框架采用双阶段串行结构，依次完成数据构建与跨域恢复建模。

首先，在数据准备阶段，对原位HRTEM视频帧进行自动筛选与划分。根据图像的空间分辨率、原子柱清晰度以及晶格完整性等质量指标，将原始图像划分为高质量（HQ）与低质量（LQ）两个子域。由于实验采集过程中无法获得同一区域的严格对齐高低质量图像，因此构建的数据集为非配对、非对齐形式。随后，将图像裁剪为固定尺寸的图像块，并根据区域属性筛选出包含纳米晶体结构的前景图像块作为训练样本，从而使网络重点学习原子排列特征，减少背景区域对模型训练的干扰。在图像恢复阶段，基于CycleGAN框架建立LQ域与HQ域之间的双向映射关系。模型包含两个生成器和两个判别器，分别学习低质量到高质量（LQ→HQ）以及高质量到低质量（HQ→LQ）的跨域转换，并通过循环一致性约束保证映射过程的稳定性与结构可逆性。与传统整图翻译方法不同，本文采用图像块级分布建模策略，在局部尺度上学习原子结构的统计分布特征，从而缩小生成的有效感受野，使网络更加聚焦于细粒度原子结构的恢复，缓解全图建模中易出现的原子错位和结构不完整问题。在此基础上，生成器采用Encoder–ViT–Decoder结构，在卷积特征提取的基础上引入自注意力机制以增强长程依赖建模能力。同时，结合空间域与频域联合特征建模机制，并通过频域一致性约束和特征分布对齐策略对生成结果进行结构保持约束，在提升图像视觉质量的同时维持原子尺度结构的物理一致性。最终，训练完成的LQ→HQ生成分支用于实际推理阶段，对低质量HRTEM图像块进行恢复，并通过滑动窗口方式重构为完整高质量图像，为后续原子柱识别与定量分析提供更加可靠的输入数据。

\subsection{空间频域联合特征提取的生成网络设计}
HRTEM-GAN以CycleGAN为骨干框架，利用两个生成器和两个判别器实现高质量（HQ）和低质量（LQ）域之间的双向映射。其中，HQ域表示在实验中筛选得到的原子柱清晰、晶格周期性完整、对比度较高的HRTEM图像子集；LQ域则对应低剂量高速采集条件下噪声占比显著、原子柱对比度衰减、局部晶格条纹断裂或模糊的退化图像子集。在实际采集过程中，高质量与低质量图像往往来源于不同时间帧或不同区域，无法形成严格配准的成对样本。因此，本研究在数据构建阶段将两类图像视为两个独立分布，在此基础上进行跨域建模。

如图\ref{fig:3-2}(a)所示，在该框架中，定义从低质量到高质量的生成器为$G_H$（即$G_{LQ \rightarrow HQ}$），其目标是将来自LQ域的输入图像映射为具有HQ域统计特征与结构表现的输出；相应地，从高质量到低质量的生成器为$G_L$（即$G_{HQ \rightarrow LQ}$），用于学习HQ域到LQ域的退化映射。与之对应的两个判别器$D_H$和$D_L$分别负责判别生成结果是否符合HQ域分布与LQ域分布。

在对抗学习框架下，判别器通过区分真实样本与生成样本进行更新。以$D_H$为例，其判别损失可表示为：

\begin{equation}
\mathcal{L}_{\mathrm{disc},H} =
\mathbb{E}_{y \sim HQ} \ell_{\mathrm{GAN}}\!\left(D_H(y), 1\right)
+
\mathbb{E}_{x \sim LQ} \ell_{\mathrm{GAN}}\!\left(D_H(G_H(x)), 0\right),
\end{equation}

其中，$\ell_{\mathrm{GAN}}$表示对抗损失函数，其形式可根据具体GAN变体选取，如二元交叉熵损失、最小二乘损失或Wasserstein目标函数等；标签1和0分别表示真实图像与生成图像。$D_L$的损失定义方式与之类似。

生成器则从三个方面进行更新：对抗损失、循环一致性损失以及身份一致性损失。以$G_H$为例，其对抗目标为：

\begin{equation}
\mathcal{L}_{\mathrm{GAN}}(G_H) =
\mathbb{E}_{x \sim LQ}
\ell_{\mathrm{GAN}}\!\left(D_H(G_H(x)), 1\right),
\end{equation}

该项损失推动生成结果在统计分布上逼近HQ域真实样本。

为保证跨域映射在非配对条件下仍能够保持结构稳定性，HRTEM-GAN通过前向循环与后向循环构建可逆约束。具体而言，给定一幅来自 $LQ$ 域的输入 $x \in LQ$，首先由生成器 $G_H$ 生成高质量风格的结果 $\hat{y} = G_H(x)$，随后再通过 $G_L$ 映射回 $LQ$ 域，得到重建结果 $\tilde{x} = G_L(\hat{y}) = G_L(G_H(x))$。此构成“$LQ \rightarrow HQ \rightarrow LQ$”的前向循环，其循环一致性损失定义为：

\begin{equation}
\mathcal{L}_{\mathrm{cyc}}(G_H,G_L) =
\mathbb{E}_{x \sim LQ}
\left\|
G_L(G_H(x)) - x
\right\|_{1}
+
\mathbb{E}_{y \sim HQ}
\left\|
G_H(G_L(y)) - y
\right\|_{1}.
\end{equation}

直观上，该约束迫使$G_H$在提升图像质量时不能任意篡改原子排列与晶格几何，而应在保留结构的前提下完成对比度恢复、噪声抑制与细节增强，从而降低“生成看似清晰但结构失真”的风险。

此外，为抑制不必要的风格漂移，当输入图像已经位于目标域时，引入身份一致性损失：

\begin{equation}
\mathcal{L}_{\mathrm{idt}}(G_H,G_L) =
\mathbb{E}_{y \sim HQ}
\left\|
G_H(y) - y
\right\|_{1}
+
\mathbb{E}_{x \sim LQ}
\left\|
G_L(x) - x
\right\|_{1}.
\end{equation}

综合上述约束，可得到模型的基础优化目标函数：

\begin{equation}
\label{eq:03:01}
\mathcal{L}_{\text{base}} = 
\mathcal{L}_{\mathrm{GAN}}(G_H, D_H)
+ 
\mathcal{L}_{\mathrm{GAN}}(G_L, D_L)
+ 
\lambda_{\mathrm{cyc}} \mathcal{L}_{\mathrm{cyc}}(G_H, G_L)
+ 
\lambda_{\mathrm{idt}} \mathcal{L}_{\mathrm{idt}}(G_H, G_L),
\end{equation}

其中，$\lambda_{\mathrm{cyc}}$和$\lambda_{\mathrm{idt}}$分别为循环一致性与身份一致性损失的权重系数。通过优化上述损失函数，网络在分布层面逼近目标域，同时在内容层面维持结构稳定性，从而实现非配对条件下低质量HRTEM图像向高质量域的可靠映射。

在完成对抗与循环约束的整体建模后，为进一步增强原子尺度结构的表达能力，本文对生成器结构进行了针对性改进。本文的方法采用由四层编码器和解码器构成的U-Net结构作为生成器主干，如图\ref{fig:3-2}(b)所示。在编码阶段，输入图像首先经过多层卷积模块逐级下采样，空间分辨率逐层降低，而通道维度逐层增加，从而实现由局部纹理到高层语义结构的特征提取。解码阶段与编码阶段结构对称，通过逐级上采样恢复空间分辨率，同时逐层减少通道数，使特征从高维语义空间逐步还原至图像空间。解码器通过融合来自上一层解码结果的上采样特征和对应编码层的跳跃连接特征，实现多尺度特征的融合与信息传递。

在编码器的每一层，本章引入如图\ref{fig:3-2}(c)所示的空间–频域交互模块（Spatial–Frequency Interaction Block），以实现双分支特征在特征层面的持续耦合与协同演化。输入特征首先在通道维度上划分为空间特征流与频域特征流：空间分支通过常规卷积模块对特征进行更新，主要学习原子柱的局部几何形态、邻域边界关系及缺陷扰动等细粒度结构信息；频域分支则通过图\ref{fig:3-2}(d)所示的频域卷积模块（Frequency Convolution Module, FCM）进行更新，其内部首先对特征执行快速傅里叶变换（FFT），将其映射至频域表示，在频谱空间中通过卷积与非线性变换对频率分量进行可学习调制，随后再通过逆傅里叶变换（IFFT）恢复至空间域。

该设计能够显式强化晶格周期性对应的主频分量与高频结构细节，从而弥补传统卷积在建模长程周期结构方面的不足。在完成各自分支的更新后，交互模块进一步引入跨分支信息交换机制：空间分支与频域分支的更新结果分别经过轻量映射后，以残差形式注入对方分支，实现双向特征校正。通过在每一层编码阶段重复执行该交互过程，两路特征在多尺度上逐层对齐并形成稳定的互补表达，为后续对抗优化与结构保持约束提供更加物理一致且频谱稳定的特征基础。

% 在完成各自分支的更新后，交互模块进一步引入跨分支信息交换机制：空间分支与频域分支的更新结果分别经过轻量映射后，以残差形式注入对方分支，实现双向特征校正。通过这种设计，空间分支在学习局部几何结构时持续受到频域周期信息的约束，从而在低信噪比条件下减少原子柱粘连与周期断裂现象；同时，频域分支在增强周期与高频成分时也受到空间几何一致性的限制，避免产生不符合局部结构的伪周期纹理或异常频谱放大。该交互过程并非简单的特征拼接，而是在每一层编码阶段重复执行，使两路特征在多尺度上逐层对齐并形成稳定的互补表达。

\subsection{基于自注意力机制的全局原子结构一致性建模}
在原子分辨率HRTEM图像中，晶格条纹与原子柱排列通常呈现出明显的周期性与长程有序性。然而，传统卷积神经网络主要依赖局部卷积算子进行特征提取，其感受野受卷积核尺寸与网络深度限制，虽然通过多层堆叠能够在一定程度上扩大感受范围，但本质上仍属于局部建模机制，难以显式刻画远距离空间位置之间的依赖关系。在低剂量成像条件下，图像信噪比下降，原子柱对比度减弱、晶格周期结构不完整，仅依赖局部卷积恢复容易导致局部清晰但整体周期失配或结构断裂的问题。因此，为增强网络对晶格长程关联性的表达能力，有必要引入能够建模全局依赖关系的机制，从而在提升图像质量的同时维持原子尺度结构的一致性。

本文在 U-Net 生成器的瓶颈层引入 Vision Transformer（ViT）结构，将卷积的局部特征建模能力与自注意力的全局依赖建模能力相结合。在编码路径完成多尺度特征提取后，将最低分辨率的高层语义特征输入至 ViT 模块进行全局关系建模，再将增强后的特征送入解码器进行重建。该设计使网络在特征恢复至高分辨率之前即可建立全局一致性约束，从而在提升图像质量的同时维持晶格结构的整体连贯性。

如图\ref{fig:3-2}(e)所示，ViT 瓶颈模块由特征序列化、线性映射、Transformer 编码器堆叠以及特征重构等步骤构成。首先，将瓶颈处融合后的二维特征图（尺寸为 $w \times h \times f$）在空间维度上展平为长度为 $w \times h$ 的 token 序列，每个 token 为维度为 $f$ 的特征向量。为保留空间拓扑结构信息，在序列中加入二维位置编码。随后，特征序列首先经过线性映射层（Linear）进行维度调整，再输入至由多层 Transformer 编码器块堆叠而成的自注意力模块。

每个 Transformer 编码器块内部结构包括：层归一化（Layer Normalization, LN）、多头自注意力（Multi-head Self-Attention, MSA）、残差连接、再次层归一化以及前馈网络（Multi-Layer Perceptron, MLP）与残差连接。其中，多头自注意力模块通过对输入特征生成查询（Query）、键（Key）和值（Value）向量，计算不同空间位置之间的相关性权重，实现全局范围内的信息聚合；随后通过残差连接与输入特征相加以稳定训练。前馈网络对注意力输出进行非线性映射与特征变换，并再次通过残差结构进行特征融合。该编码器块在瓶颈处堆叠多层，从而逐步增强特征的全局上下文表达能力。

经过Transformer模块处理后的特征序列通过线性映射恢复至原始特征维度，并重塑为二维特征图形式，尺寸仍为 $w \times h \times f$。该特征随后输入至解码路径，与编码阶段通过跳跃连接传递的局部细节特征进行融合，实现空间分辨率的逐级恢复。通过在结构瓶颈处嵌入上述ViT模块，网络在保持卷积对局部原子柱细节刻画能力的同时，显式增强了对晶格长程依赖关系的建模能力，有效抑制周期结构断裂与整体结构失配现象，提高增强结果在原子尺度上的结构一致性与物理可信度。

\subsection{基于特征函数与频域一致性的结构保持约束设计}
为了进一步增强生成结果在原子尺度上的结构保真能力，本文在对抗损失与循环一致性损失之外，引入频域一致性约束与基于特征函数（Characteristic Function, CF）的分布对齐约束，从频谱统计与特征分布两个层面对结构信息进行显式约束。

考虑到晶格结构在频域中表现为具有明显周期峰值与特征频率分布的谱结构，仅依赖空间域重建可能产生不真实的高频伪影或周期畸变。为此，本文对瓶颈层特征施加频域对齐约束。
具体而言，对生成器瓶颈特征施加快速傅里叶变换（Fast Fourier Transform, FFT），并约束其幅度谱之间的差异。设 $\mathcal{F}(\cdot)$ 表示傅里叶变换算子，$|\cdot|$ 表示复数幅值运算，则频域一致性损失定义为：

\begin{equation}
\mathcal{L}_{\mathrm{FFT}} =
\mathbb{E}_{z_H, z_L}
\left[
\left\|
\left|
\mathcal{F}(z_H(x))
\right|
-
\left|
\mathcal{F}(z_L(y))
\right|
\right\|_1
\right]
\end{equation}


其中 $z_H$ 与 $z_L$ 分别表示来自两个域的瓶颈特征表示，$\|\cdot\|_1$ 表示逐元素的 $\ell_1$ 范数。该约束鼓励生成结果在频谱统计上与目标域保持一致，从而抑制不真实的高频增强现象，并维持晶格周期结构的稳定性。

除频域对齐外，仅匹配频谱强度仍可能忽略相位信息及更高阶统计特征，为此，本文进一步引入特征函数。特征函数能够将特征分布映射至复数域，其实部与虚部分别对应余弦与正弦分量，自然支持幅度–相位分解形式。其中，幅度刻画特征分布在不同探测方向上的响应强度，反映细粒度纹理覆盖范围与局部对比变化；相位则对结构平移与几何偏移更加敏感，能够有效约束晶格畸变与原子位置漂移。

设 $\mathrm{CF}(\cdot)$ 表示特征函数映射，$|\cdot|$ 与 $\angle(\cdot)$ 分别表示复数的幅值与相位运算，则特征函数对齐损失定义为：

\begin{equation}
\mathcal{L}_{\mathrm{CF}} =
\left\|
\left|
\mathrm{CF}(z_H)
\right|
-
\left|
\mathrm{CF}(z_L)
\right|
\right\|_1
+
\lambda_{\phi}
\left(
1 - \cos
\big(
\angle \mathrm{CF}(z_H)
-
\angle \mathrm{CF}(z_L)
\big)
\right),
\end{equation}

其中 $\lambda_{\phi}$ 为相位一致性项的权重系数。第一项约束幅度分布一致性，第二项通过余弦相似度形式约束相位差异，从而在统计意义上强化结构对齐能力。

综合对抗损失、循环一致性损失以及上述结构保持约束，本文最终优化目标函数可表示为：

\begin{equation}
\mathcal{L}_{\mathrm{total}} =
\mathcal{L}_{\mathrm{base}}
+
\lambda_{\mathrm{CF}} \mathcal{L}_{\mathrm{CF}}
+
\lambda_{\mathrm{FFT}} \mathcal{L}_{\mathrm{FFT}},
\end{equation}

其中 $\lambda_{\mathrm{CF}}$ 与 $\lambda_{\mathrm{FFT}}$ 分别为特征函数约束与频域一致性约束的权重系数，用于平衡不同约束项在整体优化目标中的贡献。

通过在瓶颈特征层面联合引入频域幅度一致性与特征函数幅相对齐机制，模型能够在增强图像视觉质量的同时维持原子尺度结构的统计一致性与几何稳定性，从而有效降低结构失真风险，提高生成结果的物理可信度。

\subsection{实验数据来源及描述}
本章的数据来源于原位像差校正透射电子显微镜实验。本研究所使用的数据主要包括原位透射电子显微镜（in situ TEM）时间序列图像数据以及理论计算数据。实验数据来源于在真空环境和反应气氛条件下对金（Au）纳米晶进行的原位高分辨透射电子显微镜观测。具体而言，从原始的原位视频中提取连续图像帧作为后续原子级结构分析框架的输入数据。在真空条件下，选取了一段持续约51s 的时间分辨图像序列进行分析，该序列包含383个关键帧，帧间曝光时间为0.33s，用于定量表征表面应变的时间演化过程。在反应气氛条件下，在CO/O$_2$ 环境中采集原位TEM视频，并从中提取持续约31.5 s的图像序列，用于逐帧跟踪表面原子结构演化及应变变化。所有原位TEM实验均在 Titan ETEM G2 透射电子显微镜上完成，该设备配备像差校正成像透镜。图像及视频序列通过电荷耦合器件（CCD）相机采集，采样帧率为10帧每秒。实验所使用的样品为高纯度金丝（99.999\%，Alfa Aesar），通过原位操作制备得到用于观测的 Au 纳米晶。原始视频流通过透射电子显微镜获取后，根据图像清晰度、晶格周期完整性及原子柱对比度等标准，对单帧图像进行人工筛选，最终得到原始分辨率为1936$\times$1112像素的非配对高质量图像153张、低质量图像230张，部分数据集样本如图\ref{fig:3-3}所示。

\subsection{实验设置与评价指标}
\textbf{实验设置：}本节实验代码全部用Python语言实现，包括传统图像处理方法、评估指标等；模型训练及推理实验通过PyTorch框架实现。

在\textbf{图像重建与分类建模的训练阶段}，采用基于U-Net的编码器–解码器结构作为主干网络。其中，编码器由五个层级化的卷积模块构成，并通过下采样操作逐步提取多尺度特征；解码器则通过上采样与跳跃连接相结合的方式对空间分辨率进行对称恢复，最终输出单通道的重建结果。在重建任务方面，网络被监督以恢复输入图像块的灰度强度值，其中将 RGB 输入通过颜色通道的线性组合转换为单通道灰度图像。为强调对具有结构意义区域的恢复，仅对被标注为前景的图像块计算重建损失，而背景图像块不参与重建监督。重建目标采用重建输出与对应灰度输入之间的均方误差（MSE）损失进行建模。此外，引入了两个分类目标：其一为图像块级前景/背景分类损失，用以引导网络区分结构区域与背景区域；其二为图像级质量分类损失，用以预测图像块来源于高质量域还是低质量域。这些目标被整合为加权损失函数，其中重建损失被赋予较高权重以优先保证结构保真度，而分类损失作为辅助约束以引导特征学习。网络采用 Adam 优化器进行训练，初始学习率设为 $1\times10^{-3}$。训练批大小为16，共训练 50个epoch。在优化过程中，总损失被定义为重建损失、图像块级分类损失以及图像级分类损失的加权和，并通过最小化该总损失进行参数更新。模型选择依据验证集损失进行评估，保留验证损失最低时对应的网络参数用于推理阶段。

在\textbf{原子图像质量的训练阶段}，采用基于非配对数据进行训练，并在配备CUDA加速的NVIDIA 4090 GPU上完成训练。网络采用Adam优化器进行优化，初始学习率设为 $2 \times 10^{-4}$，$\beta_1 = 0.5$。学习率在前150个epoch内保持不变，随后在接下来的20个epoch中线性衰减至0。实验采用非对齐数据设置，并设置由低质量域（LQ）向高质量域（HQ）的翻译方向。在训练过程中，所有输入图像首先被缩放至 $286 \times 286$，随后随机裁剪为 $256 \times 256$，批大小设为 128。模型整体结构遵循标准 CycleGAN 配置，生成器与判别器的基础特征通道数均设为64，判别器采用PatchGAN结构，并基于最小二乘 GAN（Least-Squares GAN）目标函数进行训练。同时，引入大小为50的图像缓冲区（image buffer）以增强训练稳定性。

需要说明的是，尽管 HRTEM-GAN 在训练阶段采用LQ与HQ域之间的双向映射结构进行优化，但在实际应用部署时，主要使用 $LQ \rightarrow HQ$ 分支进行图像质量提升。

\textbf{评价指标：}为全面评估所提出的方法，本文从图像生成质量评估与识别性能评估两个方面开展实验。对于图像生成任务，采用定量指标衡量生成图像与真实图像之间的分布一致性，并辅以特征空间的定性可视化分析。具体而言，选用Fréchet Inception Distance（FID）\cite{heusel2017gans}与 Kullback–Leibler（KL）\cite{perezcruz2008kullback}散度进行定量评价，同时采用t-SNE可视化\cite{vandermaaten2008tsne}方法进行特征分布的定性分析。

\textbf{FID}用于计算拟合于真实图像与生成图像深度特征表示的两个多元高斯分布之间的距离，其定义为：

\begin{equation}
\mathrm{FID} = \|\mu_r - \mu_g\|_2^2 + \mathrm{Tr}\left( \Sigma_r + \Sigma_g - 2(\Sigma_r \Sigma_g)^{1/2} \right),
\end{equation}

其中，$(\mu_r, \Sigma_r)$ 与 $(\mu_g, \Sigma_g)$ 分别表示从真实图像与生成图像中提取的特征均值与协方差矩阵。FID 值越低，表示两者特征分布越接近。

\textbf{KL散度}用于进一步量化两种分布之间的差异，其表达式为：

\begin{equation}
D_{\mathrm{KL}}(P \| Q) = \sum_x P(x) \log \frac{P(x)}{Q(x)},
\end{equation}

其中，$P$ 与 $Q$ 分别表示真实样本与生成样本的特征分布。KL值越小，说明分布一致性越高。

\textbf{t-SNE可视化}用于将高维特征投影至低维空间，从而直观比较真实图像与生成图像在特征空间中的聚类分布情况。

作为对比基线，本文将HRTEM-GAN与传统去噪算法以及近年来基于学习的图像恢复方法进行比较。具体包括Wiener滤波与ABSF作为代表性的经典频域/滤波方法，以及广泛使用的基于块匹配的去噪算法BM3D。此外，还评估了三种基于深度学习的替代方法，即 AtomSegNet\cite{lin2021temimagenet}、SBD\cite{mohan2022deep}和SFIN\cite{lobato2024deep}，这些方法已在先前工作中用于提升TEM图像质量及下游原子识别任务。

为在可控条件下评估识别性能，本文采用同一分割网络（AtomSegNet）对各类恢复方法生成的图像进行原子掩膜预测，并将预测结果与真实标注进行像素级指标（Precision、Dice 与 IoU）计算，以保证评估过程的公平性与一致性。\textbf{Precision}衡量预测为前景的像素中真实为前景的比例，其定义为：

\begin{equation}
\mathrm{Precision} = \frac{\mathrm{TP}}{\mathrm{TP} + \mathrm{FP}},
\end{equation}

其中，TP与FP分别表示真阳性与假阳性像素数量。

\textbf{Dice系数}用于评估预测掩膜与真实标注之间的重叠程度，其计算公式为：

\begin{equation}
\mathrm{Dice} = \frac{2\mathrm{TP}}{2\mathrm{TP} + \mathrm{FP} + \mathrm{FN}},
\end{equation}

\textbf{IoU}定义为：

\begin{equation}
\mathrm{IoU} = \frac{\mathrm{TP}}{\mathrm{TP} + \mathrm{FP} + \mathrm{FN}},
\end{equation}

其中，FN 表示假阴性像素数量。Precision、Dice 与 IoU 值越高，表示识别性能越优。

总体而言，这些观察结果表明，所提出的方法不仅实现了常规去噪效果，而且能够生成在视觉上更接近高质量 TEM 观测结果的恢复图像。

(3)空域–频域一致性分析:

除了空域表观特征之外，衡量HRTEM修复结果是否具有物理意义的一个关键指标在于，该方法是否能够在频域中恢复特征性的晶格周期结构。为此，我们在对图像施加二维Hann窗函数之后，计算其二维快速傅里叶变换（2D FFT）的对数幅值谱，并以零频分量居中的形式进行可视化。该处理过程能够有效抑制由图像边界引起的频谱泄漏现象，从而获得更加稳定的倒易空间谱分布，用于评估与晶格相关的高频结构信息。

这些观察结果表明，HRTEM-GAN 提升图像质量的机制并非简单增强对比度，而是通过重建晶格周期结构来实现物理一致性的恢复。此外，我们还提供了具有代表性的高质量（HQ）真实图像块作为参考（并未与对应的 LQ 输入严格配对），以展示高质量图像典型的倒易空间谱特征。可以观察到，恢复结果在定性上与这些HQ参考图像的频域模式更加一致。

\section{本章小结}

本章主要针对原子分辨HRTEM图像质量提升与结构保真问题，提出了一种基于生成对抗网络的图像修复方法——HRTEM-GAN。该方法通过引入频域一致性约束与基于特征函数的分布对齐机制，在保持原子尺度结构信息的同时显著提升图像质量。实验结果表明，HRTEM-GAN在分布一致性指标（FID和KL散度）以及原子分割性能（Precision、Dice和IoU）方面均优于传统去噪方法和其他基于学习的恢复方法，验证了其在原子分辨图像修复中的有效性与优势。
